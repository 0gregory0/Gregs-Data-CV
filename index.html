NUMBER GUESSING GAME

A game where the computer picks a random integer between 1 and 100 and the user has to guess what the number is. When the user guesses a number lower than the one the computer picked, the computer informs the user that the number is too low and the user has to guess again. When the user guesses a number higher than the one the computer picked, the computer informs the user that the number is too high and the user has to guess again. When the user guesses correctly, the user is told the guess is just right and the game terminates.


I'd like to implement a feature for awarding and tracking points. I'm also thinking of adding a 'Frenzy Multiplayer' feature which will support up to four players and the computer picks multiple random numbers in different rounds and the user(s) will try to guess them. Points are rewarded based on the accuracy of the guess and the user with the highest guess wins.
